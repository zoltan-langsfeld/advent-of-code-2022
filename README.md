# advent-of-code-2022
This repository contains the solutions for the challenges found under the Advent of Code 2022 project.

## Prerequisites
* OpenJDK 17.0.5
* Apache Maven 3.8.6

## Run
In order to calculate the results, run the command below:
```
mvn exec:java
```

## Test
Run the below command to run the tests:
```
mvn test
```
